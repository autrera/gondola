# Screenshots

Real UI captures for the main [README](../../README.md) live here.

Headless capture of the running dev server is unreliable (its hot-reload socket blocks hydration), so these are best taken by hand from a running instance.

## How to add one

1. Run the app: `npm run dev`, and open the view you want.
2. Take a clean screenshot. Hide or blur anything personal (names, message content, keys).
3. Save it here using one of the names below, then embed it in the README's **Screenshots** section:

   ```markdown
   ![Semantic search](docs/screenshots/search.png)
   ```

## Suggested shots

| File | View |
| --- | --- |
| `chat.png` | Main chat with the animated presence and composer |
| `search.png` | Semantic conversation search with results |
| `models.png` | Model picker in the composer |
| `xray.png` | API X-ray with per-request cost |
| `subagent.png` | A running sub-agent task card |
| `workspace.png` | Agents, memory, and connections workspace |

Keep images reasonably sized (a 2x retina PNG under ~1 MB is ideal).
